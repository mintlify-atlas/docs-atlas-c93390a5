H2O Sonar provides **44 built-in evaluators** for LLM, RAG, and AI agent evaluation. Evaluators are organized into seven categories: Agent, Generation, Retrieval, Privacy, Fairness, Summarization, and Classification.

## Listing evaluators programmatically

Use `evaluate.list_evaluators()` to retrieve evaluator descriptors, and `evaluate.describe_evaluator()` to inspect a specific evaluator's parameters and metadata.

```python
from h2o_sonar import evaluate

# list all generative evaluators
evaluators = evaluate.list_evaluators()
for ev in evaluators:
    print(ev.id, "-", ev.display_name)

# list evaluators filtered by keyword
rag_evaluators = evaluate.list_evaluators(keywords=["evaluates_rag"])

# describe a specific evaluator
description = evaluate.describe_evaluator(
    evaluator="h2o_sonar.evaluators.rag_hallucination_evaluator.RagHallucinationEvaluator"
)
print(description)
```

<Note>
  `list_evaluators()` filters by the `llm` keyword by default, returning all generative evaluators. Pass a `keywords` list to narrow results.
</Note>

## Using evaluator IDs in a run

Pass evaluator IDs or class references to `evaluate.run_evaluation()`:

```python
from h2o_sonar import evaluate
from h2o_sonar.evaluators import rag_hallucination_evaluator, toxicity_evaluator

evaluation = evaluate.run_evaluation(
    dataset=test_lab.dataset,
    models=list(test_lab.evaluated_models.values()),
    evaluators=[
        rag_hallucination_evaluator.RagHallucinationEvaluator().evaluator_id(),
        toxicity_evaluator.ToxicityEvaluator().evaluator_id(),
    ],
    results_location="/tmp/eval-results",
)
```
