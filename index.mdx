H2O Sonar is a Python library for AI model risk management across predictive and generative systems. It provides explainers and evaluators that validate models, detect bias, assess fairness and privacy, and generate audit documentation. Built for regulated industries, H2O Sonar enables risk, compliance, and validation teams to quantify model risk and maintain robust governance throughout the model lifecycle.

## Get started in 3 steps

<Steps>
  <Step title="Install H2O Sonar">
    Download the wheel from [GitHub Releases](https://github.com/h2oai/h2o-sonar/releases) and install it with your desired extras.

    ```bash
    pip install h2o_sonar-3.2.1.whl[explainers,evaluators]
    ```
  </Step>
  <Step title="Run your first interpretation or evaluation">
    For predictive models, call `interpret.run_interpretation()`. For generative models, call `evaluate.run_evaluation()`.

    ```python
    from h2o_sonar import interpret

    interpretation = interpret.run_interpretation(
        dataset="dataset.csv",
        model=my_sklearn_model,
        used_features=feature_names,
        target_col="target",
        results_location="./results",
    )
    ```
  </Step>
  <Step title="Review your report">
    H2O Sonar generates HTML, JSON, and PDF reports in your results directory. Open the HTML report in your browser to explore explanations interactively.

    ```bash
    open ./results/h2o-sonar/mli_experiment_*/interpretation.html
    ```
  </Step>
</Steps>

## Key capabilities

| Capability | Details |
|---|---|
| **Predictive explainers** | 18 explainers: SHAP, PD/ICE, surrogate trees, fairness, drift detection, and more |
| **Generative evaluators** | 44 evaluators: hallucination, faithfulness, RAG retrieval, toxicity, privacy, and more |
| **Supported predictive models** | scikit-learn, H2O-3, Driverless AI MOJO, Driverless AI REST |
| **Supported LLM/RAG hosts** | OpenAI, Anthropic, Amazon Bedrock, Azure OpenAI, h2oGPTe, h2oGPT, ollama |
| **Report formats** | HTML (interactive), JSON (machine-readable), PDF (regulatory submissions) |
| **Platform** | Linux x86 64-bit, Python 3.11 |

</Note>
